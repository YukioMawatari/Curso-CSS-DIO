Agrupamento Seletores CSS

Podemos aplicar a mesma regra css em vários seletores diferentes. Para isso podemos separar os seletores por vírgula (,)

Assim não precisamos ficar copiando e colando os mesmos códigos para todos, nós podemos fazer um bloco CSS e reutilizar as mesmas regras
