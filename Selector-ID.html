Selector CSS

Bem-vindos

No CSS possui seletores por ID

  • Seletor por tipo busca elemntos por uma tag HTML específica
  • Seletor por ID (#) busca elementos atraves do atributo "ID"
  • Seletor por Classe (.) busca elementos através do atributo "class"
  • Seletor Universal (*) seleciona todos os elemntos do HTML
  • Seletor de atributo ([atrib]) seleciona atribulo que tem um elemento específica no nosso documento HTML, e conseguimos também buscar atribulo com um valor específico
